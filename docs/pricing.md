# Pricing Strategy

## Base Rate

The Upwork profile rate is `$35/hr`.

Use this as the default for serious AI/LLM, FastAPI, and MVP work. Quote below it only for small first-review jobs or if the job budget is explicitly lower and strategically useful.

## Adjustments by Job Type

### AI/LLM Integration

- Hourly: `$35-50/hr`
- Fixed price: minimum `$500` for meaningful scope
- Good fit: OpenAI/Anthropic API integration, RAG, AI agents, voice/chat workflows, automation pipelines
- Rationale: specialized work, higher client value, fewer qualified competitors

### FastAPI Backend / API Development

- Hourly: `$30-45/hr`
- Fixed price: based on endpoint count, integrations, database work, and deployment needs
- Good fit: REST APIs, async Python, PostgreSQL, Redis, Docker, internal tools
- Rationale: core competency and directly supported by portfolio demos

### Full-Stack MVP

- Hourly: `$30-40/hr`
- Fixed price: `$1,000-5,000` depending on scope
- Good fit: founder MVPs, internal dashboards, workflow tools, first production slice
- Rationale: combines product management background with implementation speed

### Bug Fix / Code Review

- Hourly: `$25-35/hr`
- Fixed price: `$50-200` per small issue
- Good fit: API bugs, integration failures, LLM workflow reliability, backend hardening
- Rationale: lower barrier to first hire and faster path to reviews

## Strategy for New Account

- Start at `$25-35/hr` only when the job can produce a strong first review.
- Prefer smaller jobs in the `$100-500` range if scope is clear and client history is credible.
- Avoid low-budget work that requires broad architecture ownership.
- Raise default rate after 5 strong reviews.
- Target `$50/hr` within 3 months if response rate and close rate are healthy.

## Proposal Pricing Rules

- If the client budget is strong and scope is specialized, quote `$35/hr` or higher.
- If the job is fixed-price and scope is unclear, ask for a short discovery milestone instead of guessing a full project price.
- If the job is underpriced but strategically useful, state the normal rate and optionally offer a small trial milestone.
- Do not pre-fill fixed-price bid amounts automatically; user decides after reviewing Connects, scope, and Upwork form details.

## Connects Budget

- Allocate 10-15 Connects per week during cold start.
- Prioritize jobs with payment verification, at least one hire, posted within 24 hours, and fewer than 15 proposals.
- Deprioritize jobs with no spend, vague requirements, very low fixed budgets, or high proposal counts unless fit is exceptional.
